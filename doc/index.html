Exemples de Programmes

Multiplier 6 x 8, et écrire le résultat dans R3

    mov 6 r1
    mov 8 r2
    mov 0 r3
    :loop
        sub r1 1 r1
        add r2 r3 r3
        jnez loop
        jmp endloop
    :end
        stop
    :endloop
        jmp end
